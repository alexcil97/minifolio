mi paso por el desarrollo web

Por: Alex_Cil

esto es una pagina web enfocada a visualizarme mas respecto a la busqueda de empleo, la idea es ir haciendola escalar en tecnologias, practicar con estas y poder tener ese feed positivo que buscan las empresas, como hare esto?.
Lo primero sera seguir un roadmap, el cual dejare aqui abajo:

Actualizacion: la pagina web roadmap rechaza los iframes, por lo que simplemente puse el link debajo si desean consultarlo

roadmap

siguiendo este roadmap lo que haremos sera ir viendo tecnologias mencionadas en el en orden, despues iremos actualizando esta pagina web,por ejemplo con javascript, php o python y haciendolo parte de un proyecto asi sucesivamente mas grande hasta llegar a una pagina web totalmente funcional que pase por cada tecnologia

mi periodo de aprendizaje se centrara en aprender por etapas:

primera etapa: Fundamentos web y programación

Aprende HTML y CSS - Curso Desde Cero Curso COMPLETO de JavaScript DESDE CERO para Principiantes Curso COMPLETO de GIT y GITHUB desde CERO para PRINCIPIANTES

Como habreis podido imaginar esta pagina web se compone unicamente de esta primera etapa (aunque actualmente el js basico aun no esta metido) y ha servido para poder practicar la parte de frontend

para la segunda etapa se trabajara con javascript avanzado y react, la cual nos permitira brindar de "VIDA" a nuestras paginas web, haciendolas dinamicas e interactivas, para poder aprender mejor no solo hare paginas aparte usando estas tecnologias sino que tratare de implementarlo en una web ya creada como es este caso lo cual me parece mas compelejo, ya que se puede cargar parte del trabajo ya hecho

Segunda etapa: JavaScript Avanzado y React

Curso COMPLETO de JavaScript INTERMEDIO para Subir de Nivel CURSO REACT.JS - Aprende desde cero

Ya hemos tocado un poquito en profundidad la parte del frontend, o como se le suele conocer, la cara principal, lo que ve la gente, pero ahora tocaria poder adentrarse un poquito mas no solo en como se ve, sino en como se comporta, para ello tocaremos un poco en profundida node.js y express

Tercera etapa: Backend con Node.js y Express

CURSO DE NODE.JS DESDE CERO: Introducción y primeros pasos Desarrollando una API con Express desde cero

con todo esto ya tendriamos creada nuestra primera pagina, pero no seria web, ya que solo la podriamos ver nosotros (ademas de que seguramente la hayamos creado a lo loco por que si, como es este caso) y para poder hacer que sea una pagina web de verdad necesitamos 3 cosas, la primera es que se pueda usar datos reales y deje de trabajar con datos ficticios, es decir la hace realmente funcional. La segunda es el despliegue, de nada sirve que pueda trabajar con datos reales si solo la puedes usar tu, o tu amigo en caso de que se lo pases, para ello aprender a subirlo a la web y hacerla accesible a la gente es importante y por ultimo aprender a juntar todo lo anterior, desde elegir la tematica de la pagina a implementarla y por ultimo documentarla, son pasos necesarios para todo proyecto real

Cuarta etapa: integración Full Stack

Cómo Consumir APIs en REACT como un PROFESIONAL Curso de AWS Desde Cero | Amazon Web Services 💻

para terminar necesitaremos mejorar un poco como profesionles y darnos a conocer, para ello tendremos que hacer mejora tanto de UI(User Interface)es decir todo lo que se ve y toca, como colores, tipografia, botones, disposicion de elemnetos, iconos, imagenes, etc... como de UX(User Experience)o lo que es lo mismo como el usuario siente y entiende tu applicacion a la hora de usarla, es decir que sea facil e intuitiva. y una vez terminado el ultimo retoque a la web, pasariamos a hacer nuestro portfolio profesional para que las empresas nos contraten y esas cosas.
